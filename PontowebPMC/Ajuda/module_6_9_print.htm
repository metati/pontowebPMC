Registro manual

Aprenda como registrar a frequência manualmente:
 
Na tela principal do coletor local. Vamos mostrar como efetuar os registros manualmente.
Para essa funcionalidade, você deverá ter permissão de acesso disponibilizado pelo administrador do sistema WEBPonto da TI. Siga os passos abaixo, para saber como registrar.
 
 
 

Registro Manual

 
1º Passo: Clique em registro manual vai abrir uma janela como na tela acima.
 

Usuário

 
2º Passo: Digite o CPF.
 

Senha

 
4º Passo: Clique em OK.
 

Fechar

 
A funcionalidade fechar serve para fechar o programa WEBPonto.
 
NOTA: Com isso o usuário encerra o registro do ponto manual.
