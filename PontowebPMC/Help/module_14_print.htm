Regime de Horás

 
Tela para criar um novo regime de horas.
 
1º Passo: Para criar um novo regime de hora, entre em Manuteção de frequência / Regime de horários como na imagem abaixo.
 
 
2º Passo: Nessa tela abaixo é possível visualizar todos os regimes de horas cadastrados no sistema.
 
Seta 1 - cadastrar um novo regime de hora.
Seta 2 - Alterar um regime atual
Seta 3 - Excluir um regime existente
 
 
 
 
 
 
2º Passo: Para cadastrar um novo regime de horas. Entre em Novo, abrirá a tela abaixo, preencha os campos e clique em salvar, caso queira desistir clique em cancelar
 
NOTA: Não e necessario preencher o campo Código, o mesmo preenche de forma automática.
 
 
 
 
 
 
 
 
3º Passo: Para ter certeza que o novo regime de horas foi cadastrado, entre na tela  Cadastros/ Colaborador/Usuário entre em Cadastrar Add Vínculo / regime de horas. imagem abaixo
