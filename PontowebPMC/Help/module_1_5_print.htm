Instalação do Net Frameworks
 
O frameworks padrão para rodar de forma correta e o Net. Frameworks 4.0, caso tenha outro inferior ou superior instalado no computador, será necessário remove-los e instalar o  Net. Frameworks 4.0. Siga os passos abaixo para iniciar a instalação.
 
1º Passo: Clique no arquivo dotNetFx40_Full_x86_x64 para iniciar a instalação do Frameworks.
 
 
2º Passo: Clique na opção. (Eu li e aceito os termos do contrato).
 
3º Passo: Clique em instalar para continuar.
 
 
4º Passo: Aguarde por alguns minutos a instalação dos arquivos do Net Frameworks 4.5.
 
 
 
5º Passo: Aparecendo essa mensagem abaixo a instalação foi concluída com sucesso. Clique em Concluir para finalizar a instalação.
 
 
NOTA: Siga os passos abaixo para seguir a ordem de instalação.
