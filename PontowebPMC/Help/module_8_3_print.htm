Cadastro de Colaborador

 

Como cadastrar novos funcionários.

 
No cadastro de funcionários é exibido uma lista com todos os funcionários cadastrados no sistema PontoWEB para consulta. Nela é possível inserir, alterar dados de um funcionário, bem como realizar filtros de consulta.
 
Siga os passos abaixo para iniciar um cadastro.
 
1º Passo: Para cadastrar um novo Colaborador, entre em cadastro/Colaborador.
 
2º Passo: Entre em cadastrar no canto inferior esquerdo da tela. Abrirá a tela manutenção de Colaborador.
 
NOTA: Caso queira apena alterar dados do funcionário, (Ex: Nome errado) digite o nome no campo nome completo, o mesmo será filtrado. Para localizar todos, cliques em filtrar, só o campo com o nome do setor deveram estar preenchidos. Clique em filtrar.
 
 
 
 

Manutenção de Colaborador

 

Dados Gerais

 

3º Passo: Preencha os campos como na tela abaixo. Nome, CPF, Tipo de Colaborador, Setor, Vínculo Cargo e situação.
 
A definição dos horários de trabalho deve ser executada como definido na portaria/decreto em vigência.
 
4º Passo: Em total de horas diárias, preencha os campos para definir entrada e saída do funcionário.
 
NOTA: Total de horas diárias: 8
 
Entrada Matinal: 08:00 - Saída Matinal: 12:00 - Entrada Vespertina: 14:00 - Saída Vespertina: 18:00
 
Todos os valores devem ser preenchidos, sem deixar campos em branco.
 
5º Passo: Para completar o cadastro clique em Salvar
 
6º Passo: Caso não queira completar o cadastro clique em Cancelar
 
 
 
 

Cadastrando através de Integração com sistemas de RH.

 
Caso houver a integração do sistema PontoWEB com sistema de RH, o cadastro do colaborador  será efetuado pelo gestor de forma simples rápida e eficaz. 
 
Siga os passos abaixo para iniciar um cadastro utilizando a integração do PontoWEB com sistema de RH.
 
Siga os passos abaixo para iniciar um cadastro.
 
1º Passo: Para cadastrar um novo Colaborador, entre em cadastro/Colaborador.
 
2º Passo: Entre em cadastrar no canto inferior esquerdo da tela. Abrirá a tela manutenção de Colaborador.
 
3º Passo: Preencha o campo CPF, Entre no campo superior esquerdo, como na imagem acima.
 
4º Passo: Encontrando o colaborador pelo CPF Clique no nome o sistema preenchera os campos de forma automática. EX: Tela acima.
  
5º Passo: Caso não encontrado o sistema retorna essa mensagem: Pesquisa não encontrou resultados para o CPF informado!
 
NOTA: Caso queira apena alterar dados do funcionário, (Ex: Nome errado) digite o nome no campo nome completo, o mesmo será filtrado. Para localizar todos, cliques em filtrar, só o campo com o nome do setor deveram estar preenchidos. Clique em filtrar.
 
Depois de feito o cadastro do colaborador no sistema WEB Ponto, será necessário cadastrar a biometria no sistema de coletor local. Clique aqui e aprenda como isso será possível.
