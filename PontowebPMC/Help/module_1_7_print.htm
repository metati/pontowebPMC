Configurar colaboradores no banco de dados local

Para que os Colaboradores registrem o ponto off-line, em caso de paralisação de internet, será necessário que o administrador sincronize os Colaboradores para o banco de dados local. Siga os passos abaixo para fazer a transferência dos Colaboradores.
 
 
1º Passo: Abra o coletor local (Imagem abaixo) entre em Login administrador.
 
 
2º Passo: Entre com seu login e senha, o mesmo deverá ter permissão de administrador para ter acesso às informações.
 
 
 
3º Passo: Após logar no sistema com o perfil de administrador, entre em manutenção de Colaboradores para abrir o painel principal do coletor local.
 
 
4º Passo: Ao abrir a tela principal do coletor local, entre em Carregar colaboradores.
 
 
5º Passo: Abrirá a tela carregar colaboradores, a mesma sincronizará os colaboradores para o banco de dados local. A mesma operação ocorrerá de forma automática a cada vinte e quatro horas, atualizando os novos colaboradores cadastrados no sistema PontoWEB pelo gestor do setor da Empresa/Órgão.
 
Clique em carregar para continuar o processo.
 
 
 
6º Passo: Isso poderá velar alguns minutos. Por favor, aguarde o sistema terminar o processamento!
 
 
7º Passo: Aparecendo essa mensagem os Colaboradores foram cadastrados no banco local com sucesso. Clique em Ok.
 
 
8º Passo: Ao clicar em Ok (tela acima) o processamento foi finalizado com sucesso. Cliquem em salvar e fechar para sair.
 
 
 
NOTA: Após efetuado as operações acima o sistema PontoWEB estará preparado para cumprir com todas as funcionalidades operacionais.
