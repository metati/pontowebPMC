Instalação e configuração do Banco de Dados

 
A instalação do banco de dados local será necessária para que os Colaboradores registrem a frequência em caso de não haver internet na hora do registro da frequência, os registrará Off-Line, o sistema sincronizará as informações após o retorno da internet. Siga os passos abaixo para efetuar a instalação do banco de dados local.
 
 
1º Passo:  Entre na pasta 3-Banco de Dados e Clique no arquivo SSCERuntime-ENU, será necessário definir uma pasta no computador para descompactar o arquivo de instalação.
 
 
2º Passo: Clique em Sim para descompactar o arquivo.
 
 
 
3º Passo: Clique em Browse... e defina a pasta que irá extrair o arquivo. Nesse caso o arquivo foi extraído em meus documentos teste1.
 
 
 
4º Passo: Pasta definida para à extração do arquivo Runtime. Clique em Ok para continuar.
 
 
 
5º Passo: Clique em Ok para extrair o arquivo.
 
 
 
6º Passo: Ao aparecer a mensagem abaixo clique em Ok para finalizar.
 
 
 
7º Passo: Entre na pasta escolhida para efetuar a extração do arquivo. Instale primeiro o arquivo de 32 bits SSCERuntime_x86-ENU.  Concluído o processo de instalação do arquivo de 32bits, faça o mesmo processo com o arquivo de 64bits  SSCERuntime_x64-ENU. Sempre nessa ordem.
 
 
8º Passo: Clique em Next para continuar a instalação.
 
 
9º Passo: Marque a opção como na imagem abaixo, clique em Next para continuar a instalação.
 
 
 
10º Passo: Clique em Next para continuar a instalação.
 
 
 
11º Passo: Clique em Finish para concluir a instalação.
