Cadastro de Cargo

 
Tela de cadastro de cargo.
 
No cadastro de cargo é exibido uma lista com todos os cargos cadastrados no sistema PontoWEB para consulta. Nela é possível inserir, alterar dados de um cargo, bem como realizar filtros de consulta.
 
Siga os passos abaixo para iniciar um cadastro de cargo.
 
1º Passo: Para cadastrar um novo cargo, entre em cadastro/cargo.
 
2º Passo: Ao abrir a tela de cadastro de cargo clique em cadastrar.
 
 
3º Passo: Redija o nome do cargo e clique em salvar. Pronto o cadastro do novo cargo foi efetuado.
