Relatório de Controle de Horas Extras

 
1º Passo: Após definir os usuários que farão horas extras, os mesmo poderão ser listados no relatório abaixo.
 
2º Passo: Entrar em relatórios / Relatório de controle de horas extras, como na imagem abaixo.
 
 
 
 
3º Passo: Para listar todos os usuários clique em filtrar, todos os usuários serão filtrados. A busca poderá ser feito por setor, nome ou matrícula
 
 
4º Passo: Ao acessar o menu, utilize os filtros para realizar a busca do funcionário a ser inserido as horas extras manuais. Siga os passos posteriores
 
 
NOTA: Fique atento às orientações que constam neste sistema de ajuda orientativa! Por questão de integridade, na tela acima não será possível deletar as informações. Para isso ser possível, as informações deveram ser deletadas primeiramente na tela de Lançamento de Horas Extras.
