Acesso ao ponto frequência

 
O sistema deve ser acessado pelo endereço http://pontoweb.cuiaba.mt.gov.br . Para tal, Escolha o seu navegador de preferência.
Ao acessar o PontoWEB serão solicitados o Colaborador e a senha (por patrão Login: CPF e primeira senha pontonarede em minúsculo). Os colaboradores terão seus Colaboradores previamente habilitados para o acesso ao sistema, sendo que qualquer problema de acesso deverá ser tratado por telefone ou helpdesk.
 
Perfil de acesso do Colaborador, sistema do PontoWEB:
 
O perfil de acesso atribuído ao Colaborador, define as opções disponíveis conforme a demanda solicitada pelo superintendente/coordenador do setor, considerando que cada perfil terá acesso a módulos diferentes do sistema.
 
O sistema diferencia os tipos de acesso e atribuições através do perfil de cada Colaborador. Para o PontoWEB, temos os seguintes perfis:
 
Administrador
Esse perfil possibilita o Administrador à operar todos os módulos do sistema   
 
Gestor
Esse perfil possibilita o registro do ponto e fazer as justificativas dos demais funcionários de suas atribuições no setor.
 
Colaboradores
Esse perfil possibilita que o colaboradores acesse as informações de suas atribuições de registros de ponto frequência.
