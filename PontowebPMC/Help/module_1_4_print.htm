Instalação do PontoWEB (Coletor Local)

 
Após a instalação dos componentes necessário para concluir a instalação do PontoWEB, será necessário a instalar o executável do PontoWEB e definir a permissão de acesso ao Banco de dados. Siga os passos abaixo para efetuar as duas operações.
 
1º Passo: Clique no arquivo PontoWEB pnrClientSetup, imagem abaixo.
 
NOTA: Ao instalar o executável pnrClientSetup, pode haver a necessidade o sistema pedira para instalar o NetFramewors 4.5
 
Net. Frameworks 4.5
 
 
2º Passo: Clique em Avançar para continuar.
 
 
3º Passo: O caminho a ser instado o arquivo e definido pelo sistema de forma automática. Clique em Avançar para continuar.
 
 
4º Passo: Clique em Avançar para continuar.
 
 
5º Passo: Sistema em processo de instalação. Aguarde...
 
 
 
6º Passo: Clique em Fechar para concluir. Concluído a instalação do PontoWEB será necessário configurar a permissão administrador/todos no perfil do banco de dados para o Colaborador local. Siga os passos abaixo para efetuar o procedimento.
 
 
7º Passo: Na unidade raiz do computador entre em Disco local C:\Program Files (x86)\META TI\pnrClient
 
8º Passo: Clique com o lado direito do mouse no arquivo ptServer35. imagem abaixo.
 
 
 
9º Passo: Vá até o campo propriedades do arquivo ptServer35. imagem abaixo.
 
 
 
 
 
10º Passo: Vá ate a abá segurança, selecione o campo Usuários e clique em Editar.
 
 
11º Passo: Ao clicar em Editar, selecione a novamente o campo Usuários, de permissão para todos os usuários e clique em aplicar.
 
 
 
 
 
 
 
12º Passo: Após a finalização da instação do coletor local. Clique com o lado direito do mouse no arquivo pnrClient e crie um atalho no desktop do computador. Como na imagem abaixo.
 
 
 
13º Passo: Após clicar no atalho deve renomear o mesmo com o nome PontoWEB.
