Cadastro de vínculos

Tela de cadastro de vínculo.
 
No cadastro de vínculo é exibido uma lista com todos os vínculos cadastrados no sistema PontoWEB para consulta. Nela é possível inserir, alterar dados de vínculo, bem como realizar filtros de consulta.
 
Siga os passos abaixo para iniciar um cadastro de vínculo.
 
1º Passo: Para cadastrar um novo vínculo, entre em cadastro/vínculo.
 
2º Passo: Ao abrir a tela de cadastro de vínculo clique em incluir.
 
 
 
3º Passo: Redija o nome do vínculo e clique em salvar. Pronto o cadasdro do novo vínculo foi efetuado.
